# 혈액검사 AI 해설 앱 PRD

## 1. 제품 개요
- 제품명(가칭): `Blood Insight AI`
- 목적: 혈액검사 결과를 사진 업로드, 캡처본, 수동 입력으로 받아 개인 프로필과 질환 정보를 반영해 이해하기 쉬운 인포그래픽 형태로 해설하는 Android 앱
- 지원 대상:
  - 일반 사용자: 건강검진/일반 혈액검사 해석
  - 환자: 진단받은 질환과 연관된 혈액수치 변화 해설
- 지원 플랫폼:
  - Android 11 이상

## 2. 제품 비전
- 어려운 혈액검사 결과를 누구나 이해할 수 있는 시각적 리포트로 변환한다.
- 환자에게는 질환 맥락에 맞는 설명을, 일반인에게는 건강관리 관점의 설명을 제공한다.
- AI는 "진단"보다 "설명, 정리, 질문 준비, 추세 이해"에 집중한다.

## 3. 핵심 문제
- 혈액검사 결과지는 전문 용어 중심이라 일반 사용자가 이해하기 어렵다.
- 환자는 현재 투병 중인 질환과 검사 수치의 관계를 스스로 해석하기 어렵다.
- 병원 상담 전후로 어떤 수치를 주의해서 봐야 하는지 정리하기 어렵다.

## 4. 목표
- 사용자가 혈액검사 결과를 3분 안에 업로드/입력하고 이해 가능한 요약을 받는다.
- 정상범위 대비 이상 수치를 즉시 식별할 수 있다.
- 질환 보유 사용자는 본인 질환과 관련된 중요 수치를 우선적으로 안내받는다.
- 각 항목을 눌렀을 때 의미, 주의사항, 관리 포인트를 즉시 확인할 수 있다.

## 5. 비목표
- 의사의 확정 진단을 대체하지 않는다.
- 약물 처방, 치료 변경, 응급 판단을 직접 지시하지 않는다.
- 완전 자동 OCR 결과를 검증 없이 저장하지 않는다.

## 6. 사용자 유형
### 6.1 일반 사용자
- 건강검진 결과를 이해하고 싶다.
- 높거나 낮은 수치의 의미를 쉽게 알고 싶다.
- 생활습관 관점의 관리 팁을 받고 싶다.

### 6.2 환자 사용자
- 보유 질환과 혈액수치의 관계를 알고 싶다.
- 현재 치료 중인 병이 수치에 미치는 영향이 궁금하다.
- 병원 진료 전에 어떤 수치를 질문해야 하는지 정리하고 싶다.

## 7. 핵심 기능
### 7.1 회원가입 및 로그인
- 이메일/소셜 로그인 지원
- 민감정보 동의 및 의료 고지 동의

### 7.2 사용자 프로필
- 공통 입력:
  - 성별
  - 생년월일/연령대
  - 키/체중(선택)
  - 복용약(선택)
  - 과거력/가족력(선택)
- 사용자 구분:
  - 일반 사용자
  - 환자 사용자
- 환자 추가 입력:
  - 질환명 검색
  - 병명 코드(ICD-10 또는 KCD 기반)
  - 진단 시점
  - 치료 상태(치료 중, 경과 관찰, 완치 후 추적 등)

### 7.3 검사 결과 입력
- 입력 방식:
  - 검사표 사진 업로드
  - 캡처 이미지 업로드
  - 수동 입력
- OCR 결과는 저장 전 검수 화면 필수
- 검사일, 검사 기관명, 공복 여부 등 메타데이터 입력

### 7.4 혈액검사 항목 분석
- 주요 CBC, chemistry, liver, kidney, inflammation, coagulation, iron, endocrine 항목 지원
- 정상범위 비교
- 높음/낮음/경계/정상 상태 표시
- 개인 프로필 보정
- 질환별 우선 주시 항목 강조

### 7.5 AI 해설 리포트
- 전체 요약
- 중요 항목 우선순위
- 질환 관련성 설명
- 일반적 관리 팁
- 병원에서 물어볼 질문 추천
- 응급 가능성 문구는 보수적으로 표현

### 7.6 인포그래픽 UI
- HTML 기반 리포트 렌더링
- 카드, 차트, 상태 배지, 애니메이션
- 항목 클릭 시 툴팁/바텀시트 설명

### 7.7 기록 저장 및 추세 보기
- 이전 검사 기록 타임라인
- 수치 변화 그래프
- 질환별 중요 항목 추세 강조

### 7.8 설정
- AI 공급자 선택:
  - OpenAI
  - Gemini
- API 키 직접 입력은 고급 옵션으로만 제공
- 기본 권장 구조는 서버 프록시 방식

## 8. 사용자 흐름
1. 회원가입/로그인
2. 사용자 유형 선택
3. 프로필 작성
4. 질환 정보 입력 또는 일반 건강관리 모드 선택
5. 혈액검사 결과 업로드/입력
6. OCR 검수 또는 수동 입력 확인
7. AI 분석 요청
8. 인포그래픽 리포트 확인
9. 항목별 상세 팝업 탐색
10. 기록 저장 및 추세 비교

## 9. 기능 요구사항
### 9.1 필수
- Android 11 이상 지원
- 로그인/회원가입
- 일반/환자 모드 분기
- 질환 코드 검색
- 사진/OCR/수동 입력
- 정상범위 비교
- 항목별 클릭 설명
- AI 요약 리포트
- 기록 저장

### 9.2 고급
- 검사결과 PDF 업로드
- 병원별 검사표 템플릿 인식
- 다국어 지원
- 의사 상담용 PDF 내보내기

## 10. 비기능 요구사항
- 민감정보 암호화 저장
- 분석 응답 시간 목표: 10초 이내
- OCR 검수 UI 1분 내 완료 가능
- 리포트 모바일 최적화
- 장애 시 재시도 및 부분 복구 가능

## 11. 의료/법적 안전 원칙
- 앱은 교육/설명/정보 정리 목적임을 명시
- 응급/급성 증상은 즉시 의료기관 또는 응급실 안내
- 진단/처방/치료 변경 권고는 금지
- AI 응답에는 근거 요약과 한계 고지 포함

## 12. 성공 지표
- 리포트 생성 완료율
- OCR 검수 완료율
- 항목별 상세 설명 클릭률
